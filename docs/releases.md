### [1.0.3 beta1](https://github.com/LegacyMinecraftPE/NostalgiaLauncher-Android/releases/tag/1.0.3b1) <small><i>February 1, 2025</i></small>
What's new:
- Home screen now displays a field for changing the nickname and a field for selecting versions for installation/launch
- All versions are now stored in the **Main storage**, in the NostalgiaLauncher folder
- Four previously missing versions have been added to the version list: **0.15.7.2**, **0.15.8.0**, **0.15.9.0**, **0.15.10**, and some URL errors that prevented downloads from working have been fixed
- In custom versions, you can now select your own versions from the device's storage
- and many other changes
*The app works on Android 9 - 14*
