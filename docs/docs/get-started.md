# Get started

## Installation
1. Download  the.jar file from the **[latest release](https://github.com/NLauncher/NostalgiaLauncherDesktop/releases/latest)** to a folder that does NOT contain spaces, special characters, or Cyrillic characters
2. Double click on the downloaded .jar file, or use the command in the terminal `java -jar filename.jar`

**NostalgiaLauncher Desktop works ONLY on Windows and Linux.**
